2.Identify void elements and how they are denoted
There is a group of elements that are exceptions to this rule.
These elements are called empty or void and only have a start tag since they can't have any content. Void elements can have attributes

3.Identify attributes
Entities contain attributes, which are characteristics or modifiers, qualities, amounts, or features.
An attribute is a fact or nondecomposable piece of information about an entity.

Css

1.Name the components of a css rule properly
CSS rule-set consists of a selector and a declaration block: The selector points to the HTML element you want to style.
The declaration block contains one or more declarations separated by semicolons.
Each declaration includes a CSS property name and a value, separated by a colon

2.Use combinators properly: direct child, descendant, adjacent sibling, sibling
Web content refers to the textual, aural, or visual content published on a website.
Content means any creative element, for example, text, applications, images, archived e-mail messages, data, e-services, audio and video files, and so on.
Web content is the key behind traffic generation to websites.

3.Specify colors by color names and in hexadecimal notation
yellow = #FFFF00, blue = #0000FF, silver = #C0C0C0, Purple = #800080

4.Declarations for color, background color
background-color: #0000FF; oder background-color: blue;

5.Pseudo classes for the a element
A pseudo-class in CSS is a key word placed behind a selector to query a particular state. (hover, visited, checked,...)

6.Declaration for background image

10.Identify the difference between serif and sans-serif fonts
Serif: Serif fonts have small lines at the ends on some characters
Sans-serif: "Sans" means without - these fonts do not have the lines at the ends of characters
